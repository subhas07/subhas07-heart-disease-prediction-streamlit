# Heart Disease Prediction using Machine Learning

## Overview
This project predicts heart disease risk using Machine Learning and Streamlit.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## ML Model
- KNN Classifier

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
